# Gwii EBNF

```ebnf
모든 문자 = ? 모든 유니코드 스칼라 값 ?;
공백 문자 = ? 유니코드에서 공백으로 분류된 문자 ?;
공백 = {공백 문자};
이름 문자 = ? 공백, 특문을 제외한 문자 ?;
이름 = 이름 문자, {이름 문자};
조사 = 이름 (* 더 엄밀한 정의가 필요함 *);
타입 = 이름 (* 더 엄밀한 정의가 필요함 *);
조사 붙은 이름 = 이름, "/", 조사;
파라미터 = [타입 "인"], 조사 붙은 이름;
동사 선언 =
  ["외부"], 공백, "동사", 공백, ":", 공백, {파라미터, 공백}, "`", 이름, "`", 공백,
  ["에서", 공백],
  "{", {구문} "}";
고정값 = 정수 고정값 | 실수 고정값 | 문자 고정값 | 문자열 고정값 | 진리 고정값;
정수 고정값 = (
  "1"..."9", {"0"..."9"} |
  ("02:" | "0b"), {"0" | "1"} |
  ("08:" | "0o"), {"0"..."8"} |
  "016:", {"0".."9" | "가"..."바" | "a"..."f" | "A"..."F"} |
  "0x", {"0".."9" | "a"..."f" | "A"..."F"}
);
문자 고정값 = "'" 모든 문자 "'"; (* 이스케이프 지원 *)
문자열 고정값 = "\"" {모든 문자} "\""; (* 이스케이프 지원 *)
진리 고정값 = "참" | "거짓";
```
